# Smart To-Do List

A desktop task management application built with Python and Tkinter.

---

## Description

This project is a Smart To-Do List app that helps you manage your daily tasks.
You can add tasks, set their priority, mark them as done, and delete them.
All tasks are saved in a JSON file so they remain after closing the app.

---

## Features

- Add new tasks
- Set priority level (High / Medium / Low)
- Mark tasks as Done
- Delete tasks
- Auto-save tasks to a local JSON file
- Tasks sorted by priority automatically

---

## Technologies Used

- Python 3
- Tkinter (GUI)
- JSON (data storage)

---

## How to Run

1. Make sure Python 3 is installed on your computer
2. Download the project files
3. Open the terminal and run:

```
python todo_student.py
```

---

## Project Structure

```
smart-todo-list/
│
├── todo_student.py   # main application file
├── tasks.json        # tasks data file (created automatically)
└── README.md         # project description
```

---

## How to Use

1. Type your task name in the input box
2. Choose the priority from the dropdown menu
3. Click **Add Task** to add it
4. Click on a task then click **Mark as Done** to complete it
5. Click on a task then click **Delete Task** to remove it
